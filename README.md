# Password Manager

A secure and user-friendly password manager application built with JavaScript. Safely store, manage, and access your passwords with encryption and a clean interface.

## 🌟 Features

- **Secure Password Storage** - Encrypt and securely store your passwords
- **Password Generation** - Generate strong, random passwords
- **Search & Filter** - Quickly find saved passwords
- **Copy to Clipboard** - One-click password copying
- **User-Friendly Interface** - Clean and intuitive design
- **Cross-Platform** - Access from any device via the web

## 🚀 Live Demo

Check out the live application: [Password Manager](https://password-manager-azure-ten.vercel.app)

## 🛠️ Technologies Used

- **Frontend:** JavaScript, HTML, CSS
- **Deployment:** Azure/Vercel
- **Storage:** Browser local storage / Cloud database

## 📋 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/ShubhamTiwari007xx/password-manager-.git
cd password-manager-
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open your browser and navigate to `http://localhost:3000`

## 📖 Usage

1. **Add Password:** Click the "Add Password" button and enter your credentials
2. **View Password:** Click on any entry to view the full password
3. **Generate Password:** Use the password generator to create strong passwords
4. **Delete Password:** Remove passwords you no longer need
5. **Search:** Use the search bar to find specific passwords

## 🔒 Security

- All passwords are encrypted and stored securely
- Never share your master password
- Clear browser cache regularly for added security
